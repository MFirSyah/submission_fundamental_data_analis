# Olist E-Commerce Data Analysis Dashboard 🛒✨

Dashboard interaktif ini menyajikan analisis mendalam terhadap **Brazilian E-Commerce Public Dataset by Olist**. Proyek ini mencakup seluruh siklus analisis data mulai dari *Data Wrangling*, *Exploratory Data Analysis* (EDA), hingga pembuatan visualisasi data untuk menjawab tantangan bisnis strategis.

## 📊 Ringkasan Analisis
Dashboard ini fokus pada dua masalah utama:
1. **Analisis Logistik**: Pengaruh *Shipping-to-Price Ratio* terhadap kepuasan pelanggan di wilayah Northeast dan North Brazil[cite: 1].
2. **Analisis Pembayaran**: Perbandingan durasi validasi pembayaran antara metode 'Boleto' dan 'Credit Card' selama periode Flash Sale (Mei-Juni 2018)[cite: 1, 2].
3. **Analisis Lanjutan**: Segmentasi pelanggan menggunakan metode **RFM** (Recency, Frequency, Monetary) dan analisis **Geospasial**[cite: 1].
